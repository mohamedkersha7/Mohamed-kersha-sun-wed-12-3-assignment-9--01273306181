# ContactHub

**ContactHub** هو مشروع ويب لإدارة جهات الاتصال (CRUD) مع تسجيل دخول. يتيح لك إنشاء جهات الاتصال، تعديلها، حذفها، والبحث فيها بسهولة. كما يدعم تمييز جهات الاتصال كمفضلة أو طارئة وإضافة صور شخصية لكل جهة اتصال.

## المميزات
- تسجيل دخول المستخدمين.
- إنشاء جهات اتصال جديدة (Add Contact).
- تعديل جهات الاتصال الحالية (Update Contact).
- حذف جهات الاتصال (Delete Contact).
- البحث في جهات الاتصال (Search).
- تمييز جهات الاتصال كمفضلة أو طارئة.
- عرض جهات الاتصال في قوائم جانبية للمفضلة والطوارئ.
- رفع صور لكل جهة اتصال أو استخدام صورة افتراضية.
- تحقق من صحة البيانات (Validation) مثل رقم الهاتف المصري والبريد الإلكتروني.

## التقنيات المستخدمة
- HTML5
- CSS3 (Bootstrap)
- JavaScript (Vanilla JS)
- LocalStorage لتخزين البيانات محليًا
- SweetAlert لتأكيد العمليات (حذف، إضافة، تعديل)

## طريقة الاستخدام
1. افتح المشروع في المتصفح.
2. قم بتسجيل الدخول.
3. استخدم الفورم لإضافة جهات الاتصال.
4. استخدم أيقونات التحرير، الحذف، أو التمييز بالمفضلة والطوارئ حسب الحاجة.
5. يمكن البحث عن جهة الاتصال باستخدام صندوق البحث العلوي.

## ملاحظات
- المشروع يعمل بالكامل على المتصفح ولا يحتاج إلى قاعدة بيانات خارجية.
- جميع البيانات محفوظة محليًا في LocalStorage.
